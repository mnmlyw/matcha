# Roadmap

## Editor
- Multiple cursors (Cmd+D select next occurrence, Cmd+click add cursor)
- Buffer word completion (Tab/Esc to complete from words in file)
- Indent guides (faint vertical lines at tab stops)
- Minimap (scrollbar-width code overview)
- File watcher (detect and reload external changes)
- Soft tabs (backspace deletes tab-width spaces at once)
- Code folding

## Navigation
- Command palette (Cmd+Shift+P)
- File tree sidebar
- Recent files menu
- Fuzzy file finder (Cmd+P)
- Symbol outline / go to symbol

## Languages
- Tree-sitter integration (accurate parsing, folding, selection)
- LSP client (autocomplete, diagnostics, go to definition)
- More languages: HTML, CSS, Ruby, Elixir, Haskell, OCaml, Kotlin, Dart, SQL

## UI
- Custom theme loading from JSON files
- Smooth scroll animation
- Split panes (horizontal/vertical)
- Welcome screen with recent files
- Drag to reorder tabs

## System
- Auto-save / crash recovery
- Sparkle auto-update
- Universal binary (arm64 + x86_64)
- Homebrew cask distribution
